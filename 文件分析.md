# Operit-1.9.1 项目文件分析

## 项目概述

**项目名称**: Operit AI - 智能助手应用  
**版本**: 1.9.1  
**平台**: Android 8.0+ (API 26+)  
**开发语言**: Kotlin, Java, C++, JavaScript/TypeScript  
**构建工具**: Gradle (Kotlin DSL)  
**架构**: MVVM + Jetpack Compose  

### 核心功能
- AI对话系统（支持OpenAI、Claude、Gemini、DeepSeek、通义千问等）
- 本地模型推理（MNN、llama.cpp）
- UI自动化（Accessibility Service、ADB、Root、Shizuku）
- Ubuntu 24终端环境
- 脚本引擎（JavaScript/TypeScript）
- MCP协议集成
- 工作流系统
- 记忆库系统
- 角色卡系统

## 总结

Operit-1.9.1是一个功能完备的Android AI助手应用，具有以下特点：

1. **模块化架构**: 清晰的模块划分，便于维护和扩展
2. **多模型支持**: 集成多种AI模型（云端+本地）
3. **强大自动化**: 三通道UI自动化（Accessibility/ADB/Root）
4. **完整工具链**: 从文件处理到网络请求的完整工具集
5. **脚本生态**: JavaScript/TypeScript脚本引擎，支持自定义工具
6. **高性能**: KMP流式渲染，优化内存和性能
7. **跨平台**: 支持多种Android设备和权限模式

项目代码结构清晰，文档完善，适合学习和二次开发。
